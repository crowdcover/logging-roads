The Logging Roads Initiative is a joint project of the World Resource Institute and CrowdCover that aims to analyze and make available geographic data on all logging roads in the Congo Basin region of Central Africa. Using fifteen years of satellite imagery, the project will record the precise spread of logging roads over time. All data will be added to the OpenStreetMap and licensed in the public domain.

If you would like to contribute, please read the project guides on interpreting satellite imagery and using the editing tools. To learn more about OpenStreetMap and open geographic data, see our About page.
